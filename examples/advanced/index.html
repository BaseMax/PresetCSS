Prefix. Minify. Extend.

This preset adds prefix = "u-", custom fonts, semantic spacing tokens, and per-group variant opt-in. Every generated class becomes u-{class}.

extend.spacing — semantic tokens

[extend.spacing] adds to the default scale without replacing it. Semantic names like hero produce u-p-hero, u-mt-hero, etc.

u-p-18 / u-mt-18 → "18" = "4.5rem"
u-p-hero / u-mt-hero → hero = "22rem"

Using u-card + u-flex + u-font-sans + u-font-bold. All prefixed. All from a single preset.toml.

preset.toml

# Slightly advanced preset: extend, custom utilities, per-group variants.
[options]
output = "public/app.css"
content = ["templates/**/*.html"]
minify = true
base = "minimal"
prefix = "u-" # all classes become .u-bg-primary etc.
source_comments = false

[colors]
primary = "#3b82f6"
primary-dark = "#1d4ed8"
danger = "#dc2626"

[extend.spacing]
"18" = "4.5rem"
hero = "22rem"

[fonts]
sans = "Inter, system-ui, sans-serif"
mono = "JetBrains Mono, monospace"

[breakpoints]
tablet = "768px"
desktop = "1100px"
wide = "1400px"

[variants] # opt-in, per group — no wasted CSS
colors = ["hover", "focus"]
spacing = []
static = ["responsive"]
utilities = ["hover", "responsive"]

[utilities] # escape hatch: raw CSS, still no JavaScript
btn = "padding: 0.5rem 1rem; border-radius: 6px; background: #3b82f6; color: white;"
card = "background: white; border-radius: 8px; box-shadow: 0 1px 3px rgb(0 0 0 / 0.1);"
