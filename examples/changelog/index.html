Changelog

All notable changes to PresetCSS. Follows Semantic Versioning.

Jun 2026

v0.1.0 Initial release

First public release of PresetCSS. One flat TOML file → optimized utility CSS.

new Full parse → resolve → generate → emit pipeline in presetcss-core
new CLI with build, watch, init, validate, explain, and export commands
new Token groups: colors, spacing, sizes, fonts, text, radius, shadows, breakpoints
new Static utility classes: flex, grid, text alignment, positioning, and more
new Per-group variant opt-in via [variants]
new Aho-Corasick content scanning for tree-shaking
new WASM playground for browser-based preview

May 2026

v0.1.0-rc.2 Release candidate 2
fix Fixed class collision detection not triggering on [extend.*] tables
fix CSS escaping for class names containing special characters
change Renamed output_path option to output

Apr 2026

v0.1.0-rc.1 Release candidate 1
new Minification support via minify = true
new prefix option — all generated classes get a custom prefix
breaking Removed [options] theme in favour of top-level token tables

Feb 2026

v0.1.0-beta Initial beta

First internal beta. Core pipeline stable. CLI commands working. Playground prototype complete.
